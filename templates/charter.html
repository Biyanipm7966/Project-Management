Project Description

The Punch Card project aims to develop a user-friendly, cloud-based system to replace the outdated UC Flex system used by student workers to clock in and out of campus jobs. The new system will enhance user experience, improve efficiency, and ensure reliability.

Busienss Case

UC Flex is currently considered cumbersome and inefficient by users. Punch Card will create a smoother, easier to use system for student workers, improving daily operations and reducing frustrations. This project supports better workforce management on campus and aligns with the institution's goals of operational efficiency and student satisfaction.

Expected Goals / Deliverables

  • Improve Efficiency: Streamline the clock-in and clock-out process for student workers.
  • Enhance User Experience: Develop an intuitive user interface that simplifies the process for end users.
  • Reliable System: Ensure that the new system is cloud-based to reduce the reliance on local hardware and minimize system outages.
